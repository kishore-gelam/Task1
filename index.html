What is this?

The this keyword in JavaScript refers to the object that is executing the current function. Its value depends on how a function is called.

Example: Object Method

const person = {
  name: "John",
  greet: function () {
    return this.name;
  }
};

console.log(person.greet());

In Global Scope

In the global scope, this refers to the window object.

In Arrow Functions

Arrow functions do not have their own this.

In Event Handlers

this refers to the element that received the event.
